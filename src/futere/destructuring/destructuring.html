Actividad 6: Destructuring Objects

Descripción: El destructuring es una expresión de JavaScript que permite desempacar valores de arreglos o propiedades de objetos en distintas variables.

Objeto Persona
const person = {
  name: 'Juan Carlos Valencia',
  age: 30,
  city: 'Cali',
  profession: 'Desarrollador'
};
